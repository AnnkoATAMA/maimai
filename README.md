- 必要なもの

## gitの使い方の注意点
### 作業前
- git branchで自分のブランチを絶対確認しよう。
- main,developでは基本的に作業せず、 feature/~~機能ごとでチェックアウトして作業してね
- 作業前にpullしてコンフリクト避けよう！
### 作業後
- プルリク出してmargeはコードレビューするまで絶対にしないでね

### よく使うコマンド
```
git checkout 移動したいブランチ
```
```
git add 変更した場所
git commit -m "コミットメッセージ"
git push origin 作業ブランチ
```
